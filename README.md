# Gaining ALL items in BLoxd

Copy and Paste the code in ```wordcode.js``` to your world code. 

Typed the command ```/allblocks```, the recepie for all the blocks (currently new at December 12, 2025) will be added to the recepie in the crafting table (a.k.a workbench). 

Type the command ```/allitems```, the recepie for all the items (current new at December 12, 2025) will be added to the recepie in the crafting table (a.k.a workbench).

##### IMPORTANT NOTE! You need to have a stone in inventory to "craft" items and blocks, and you need to throw them on the ground and then pick up to be able to ACTUALLY place it.

### Note: Banned List

The banend list contains substrings that the program filters out to reduce the amount of repetitive items.
